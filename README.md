# Controls
W/A/S/D: Move camera Dorward, Left, Backwards and Right.

R/F: Move camera Up/Down.

Hold Right Mouse Button and move mouse: Move camera.

Scroll wheel: Zoom in/out.

Hold Shift: Double camera speed.

Esc: Quit.
